Service Info.

Home page

1) moment를 활용해 오늘 추가된 todo 및 memo 가져오기

2) 왼쪽 메뉴 클릭시 해당 컨텐츠 노출

3) light mode와 dark mode에 따른 메뉴와 전체 컨텐츠 색상 변동

4) 오른쪽 상단 현재 날짜 노출

Todo page

1) redux(createSlice)로 상태관리

2) CRUD구현

3) useEffect를 이용해 조건문 생성 -> 해당 탭에서의 현재 완료항목 배열을 가져와서 있을경우 항목을 나열, 없을 경우 notfound 이미지 노출

4) 할일 추가시 기존 항목과 중복 체크 (수정완료된 항목 포함) -> 중복시 alert창 띄우고 return

5) 항목 수정시 내용이 없다면 alert창 띄우고 return

6) 완료 체크시 최종적으로 수정된 내용으로 완료항목 이동

7) 완료 체크시 redux와 useState를 이용해 해당 항목에 취소선 생성과 아이콘 style삭제되고 수정 및 삭제 버튼 비활성화

8) moment를 활용한 모든 할일에 추가 또는 수정한 date 노출

memo page

1) redux(createSlice)로 상태관리

2) CRUD구현

3) '+'버튼 클릭시 메모장 생성됨, 해당 메모장 클릭시 내용 수정 가능

4) 각 메모장의 배경 색상 선택 가능

5) 각 메모장의 모든 내용은 최종 수정된 내용으로 유지
